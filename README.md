# 🤖 Я с тобой, я рядом — Telegram-бот

Мягкий ИИ-спутник для девушек. Не психолог — просто тот, кто всегда рядом.

## Что умеет

- 💬 Просто поговорить — выслушать, поддержать
- 🌅 Утренний ритуал — мягкое начало дня
- 🌙 Вечерний ритуал — тихий вечер
- 📊 Дневник настроения — от 1 до 5
- 💛 Поддержка проекта — донаты

## Установка

### 1. Клонировать или скачать файлы

### 2. Установить Python 3.10+

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения
```bash
cp .env.example .env
# Отредактируйте .env и вставьте свои ключи
```

### 5. Запустить
```bash
python bot_ya_ryadom.py
```

## Получение ключей

### Telegram Bot Token
1. Найдите @BotFather в Telegram
2. Отправьте /newbot
3. Следуйте инструкциям
4. Скопируйте токен

### DeepSeek API Key
1. Зарегистрируйтесь на platform.deepseek.com
2. Пополните баланс (минимум $5)
3. Создайте API Key в разделе API Keys
4. Скопируйте ключ (показывается один раз!)

## Безопасность

⚠️ **Никогда не публикуйте свои API-ключи!**
- Не отправляйте в чаты
- Не выкладывайте на GitHub
